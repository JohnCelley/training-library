# Training Module Library (GitHub Pages)

A simple, static site you can publish on GitHub Pages. Add or edit modules by updating `modules/modules.json`.

## Publish on GitHub Pages
1. Create a **public** repo on GitHub.
2. Upload this folder to the repo (or push via git).
3. In the repo: **Settings → Pages** → Source: `Deploy from a branch`, Branch: `main`, Folder: `/ (root)`.
4. Wait for the site URL to appear; then share it.

## Customize
- Colors & spacing in `assets/style.css`
- Behavior & filters in `assets/app.js`
- Page copy in `index.html`, `SOP.html`, `README.html`
