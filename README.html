What is this?

A super-lightweight library you can host on GitHub Pages. All content lives in modules/modules.json.

How to publish

  1. Create a new public repo on GitHub.
  2. Upload this folder to the repo.
  3. Go to Settings → Pages and set the Source to Deploy from a branch, branch main, folder /.
  4. Wait for it to build; your site will be live at the URL GitHub provides.

Customize

  • Edit assets/style.css for colors and spacing.
  • Edit assets/app.js to tweak filters, fields, or card layout.
  • Replace the logo/name in the header with your own.

Made for fast updates by editing a single JSON file.
